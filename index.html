SlideTab by flyworld

A jQuery plugin providing sliding tabs and contents.

Usage

Add SlideTab css to your html file

Load jQuery, and the SlideTab plugin: 

			<script type="text/javascript" src="jquery.min.js">		
			<script type="text/javascript" src="jquery.slidetab-1.0.min.js">

Add HTML like following DOM structure 

			

				
    
					
  • 
						Tab1
					
    • 
					
  • 
						Tab 
     2    
					
    • 
				

				

					
    
						
  • 
							Content 1
						
    • 
						
  • 
							Content 2
						
    •

Activate SlideTab on outer DIV 

			$("#mySlideMenu").slideTab();
